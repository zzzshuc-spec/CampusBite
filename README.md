# 大学城美食地图

面向学生群体的本地化美食发现 Web 应用，帮助解决“吃什么”的决策困难。  
项目基于 `Next.js + TypeScript + Tailwind` 构建，内置 4 所高校、每校 10 家餐厅示例数据。

## 功能亮点

- 高校维度切换：不同高校有独立商圈与餐厅数据
- 地图探索：标注查看 + 列表联动 + 多条件筛选
- 智能决策：推荐分算法 + “不知道吃什么”随机抽取器
- 餐厅详情：招牌菜、步行时间、标签场景、营业信息
- 本地状态：收藏、最近浏览、最多 3 家餐厅对比
- 学生场景增强：省钱榜、深夜食堂、考试周安静就餐
- 深浅色主题切换、移动端优先布局

## 运行方式

```bash
npm install
npm run dev
```

访问 [http://localhost:3000](http://localhost:3000)

## 脚本

```bash
npm run lint      # 代码检查
npm run test      # 运行 Vitest 单测
npm run test:watch
```

## 项目结构

- `src/app`：页面路由（首页、地图页、详情页）
- `src/components`：地图、筛选、卡片、对比、头部组件
- `src/data`：高校和餐厅 seed 数据
- `src/lib`：筛选、推荐、步行时长估算等核心逻辑
- `src/store`：用户偏好状态与本地持久化
- `src/types`：领域类型定义
